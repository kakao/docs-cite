# cite documentation

## usage
* get theme
    ```
    git submodule init
    git submodule update
    ```
    
* hugo server
    ```
    $ hugo server
    ```

* publish
    ```
    $ ./publish.sh
    ```

## reference
* hugo : https://gohugo.io/
* hugo hosting on github pages : https://gohugo.io/tutorials/github-pages-blog/
